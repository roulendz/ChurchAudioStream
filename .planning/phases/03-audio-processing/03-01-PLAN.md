**processing-types.ts** -- Define interfaces for all processing config:

1. `AudioModeType` -- literal union: `"speech" | "music"`

2. `AgcConfig` interface:
   - `enabled: boolean` (default true -- bypass toggle)
   - `targetLufs: number` (default -16, range -20 to -14)
   - `maxTruePeakDbtp: number` (Speech: -2, Music: -1 -- derived from mode, see below)

3. `OpusEncodingConfig` interface:
   - `enabled: boolean` (default true -- bypass toggle)
   - `bitrateKbps: number` (default 128, range 48-192)
   - `frameSize: 10 | 20 | 40` (default 20)
   - `fec: boolean` (default false)
   - `dtx: false` (always false, no DTX per user decision)
   - `bitrateMode: "vbr" | "cbr"` (default "vbr" -- maps to constrained-vbr in GStreamer)
   - `audioType: "voice" | "generic"` (derived from mode: Speech->voice, Music->generic)

4. `RtpOutputConfig` interface:
   - `rtpPort: number`
   - `rtcpPort: number`
   - `host: string` (always "127.0.0.1")
   - `ssrc: number` (unique per channel, generated from channel index)

5. `ProcessingConfig` interface (the top-level per-channel processing config):
   - `mode: AudioModeType` (default "speech")
   - `agc: AgcConfig`
   - `opus: OpusEncodingConfig`
   - `rtpOutput: RtpOutputConfig`

6. `ProcessingDefaults` -- exported const with all default values for a Speech-mode channel.

7. `deriveSettingsFromMode(mode: AudioModeType, current: ProcessingConfig): ProcessingConfig` -- pure function that updates mode-dependent fields when mode changes:
   - Speech: opusenc audioType = "voice", agc maxTruePeakDbtp = -2, opus bandwidth implicitly wideband (let GStreamer choose)
   - Music: opusenc audioType = "generic", agc maxTruePeakDbtp = -1, opus bandwidth implicitly fullband (let GStreamer choose)
   - Preserves all other settings (bitrate, fec, target lufs, bypass flags, frame size, etc.)

**port-allocator.ts** -- Pure function module:

1. `const RTP_BASE_PORT = 77702`

2. `getPortsForChannel(channelIndex: number): { rtpPort: number; rtcpPort: number }` -- returns `{ rtpPort: RTP_BASE_PORT + (channelIndex * 2), rtcpPort: RTP_BASE_PORT + (channelIndex * 2) + 1 }`

3. `generateSsrc(channelId: string): number` -- deterministic SSRC from channel ID. Use first 4 bytes of a simple hash of the channel UUID, ensuring non-zero. This guarantees unique SSRC per channel without random collisions.

**schema.ts** -- Add processing Zod schemas ABOVE the ChannelSchema definition:

1. `AgcSchema = z.object({ enabled: z.boolean().default(true), targetLufs: z.number().min(-20).max(-14).default(-16), maxTruePeakDbtp: z.number().min(-6).max(0).default(-2) })`

2. `OpusEncodingSchema = z.object({ enabled: z.boolean().default(true), bitrateKbps: z.number().int().min(48).max(192).default(128), frameSize: z.enum(["10", "20", "40"]).default("20"), fec: z.boolean().default(false), bitrateMode: z.enum(["vbr", "cbr"]).default("vbr") })`
   NOTE: Store frameSize as string enum in Zod for JSON serialization, convert to number at pipeline build time.

3. `RtpOutputSchema = z.object({ rtpPort: z.number().int().min(1024).max(65534).default(77702), rtcpPort: z.number().int().min(1025).max(65535).default(77703), ssrc: z.number().int().min(1).default(1) })`

4. `ProcessingSchema = z.object({ mode: z.enum(["speech", "music"]).default("speech"), agc: AgcSchema.default(() => AgcSchema.parse({})), opus: OpusEncodingSchema.default(() => OpusEncodingSchema.parse({})), rtpOutput: RtpOutputSchema.default(() => RtpOutputSchema.parse({})) })`

5. Extend `ChannelSchema` to include: `processing: ProcessingSchema.default(() => ProcessingSchema.parse({}))`
