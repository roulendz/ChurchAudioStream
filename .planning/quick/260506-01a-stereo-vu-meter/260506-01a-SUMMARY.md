# Quick Task 260506-01a: Stereo VU Meter — Summary

## What changed

### VuMeter.tsx (admin canvas meter)
- Added `channelCount?: 1 | 2` prop (default 1, backward compatible)
- Extracted `drawBar()` helper for single-bar canvas rendering (SRP/DRY)
- Changed `stateRef` from single `MeterState` to `MeterState[]` array
- Stereo mode renders two side-by-side bars with 2px gap, each with independent smoothing/peak-hold
- Graceful fallback: if levels array shorter than channelCount, duplicates first channel value

### VuMeterBank.tsx (monitoring page)
- Added `outputFormat: string` to `ChannelInfo` interface
- `VuMeterItem` passes `channelCount` derived from `outputFormat === "stereo"`

### ChannelCard.tsx (admin channel card)
- Passes `channelCount` and adjusts width (32px stereo, 24px mono) to VuMeter

### LevelMeter.tsx (listener HTML meter)
- Added `channelCount?: 1 | 2` prop (default 1)
- Extracted `MeterColumn` component for single-column bar rendering (DRY)
- Stereo mode renders two side-by-side columns with 2px gap
- Component remains unused in listener views (wiring is a separate task)
